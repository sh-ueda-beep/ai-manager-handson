記録済みのすべてのハンズオンステップを1つのMarkdownチュートリアルドキュメントに結合し、エクスポートする。

---

## 動作フロー

### 1. ステップファイルの取得

`handson/steps/` ディレクトリ内の `step-*.md` ファイルをステップ番号順に取得する。

```bash
ls -1 handson/steps/step-*.md 2>/dev/null | sort
```

**ステップが存在しない場合：**

```
エクスポートするステップがありません。
まず `/handson:record` でステップを記録してください。
```

ここで処理を終了する。

### 2. 各ステップの読み取り

各ステップファイルを読み取り、フロントマターと本文を分離する。

### 3. チュートリアルドキュメントの生成

以下のフォーマットで `handson/exports/tutorial-full.md` を生成する：

```markdown
# ハンズオン チュートリアル: OpenSpecを使ったプロジェクト管理

> このチュートリアルは自動生成されました。
> 生成日時: {現在の日時}

## 目次

1. [{ステップ1のタイトル}](#step-1)
2. [{ステップ2のタイトル}](#step-2)
...

---

## ステップ {番号}: {タイトル} {#step-{番号}}

> ステータス: {status} | 作成日: {created} | タグ: {tags}

{ステップの本文（フロントマターを除く）}

---

（以降、全ステップを繰り返し）

---

## まとめ

このチュートリアルでは以下のステップを実施しました：

1. {ステップ1のタイトル}
2. {ステップ2のタイトル}
...
```

### 4. ファイルの書き出し

生成したドキュメントを `handson/exports/tutorial-full.md` に書き出す。

### 5. 完了メッセージの表示

```
## チュートリアルをエクスポートしました

- 出力先: `handson/exports/tutorial-full.md`
- ステップ数: {N}
- ステータス内訳: draft: {n1}, review: {n2}, final: {n3}

ファイルを確認してください。
```

---

## ガードレール

- 既存の `tutorial-full.md` は上書きする（エクスポートは再実行可能）
- ステップ番号順にソートして出力する
- フロントマターはエクスポート先には含めず、メタ情報は引用ブロックで表示する
- すべてのテキストは**日本語**で記述する
